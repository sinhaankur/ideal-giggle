# Contributing to this project

Thank you for your interest in contributing. This project is open source but changes are only merged after review and explicit approval.

## How to contribute

- **1. Fork the repository** on GitHub (`sinhaankur/ideal-giggle`).
- **2. Create a feature branch** from `main` in your fork.
- **3. Make your changes** with clear, focused commits.
- **4. Open a Pull Request (PR)** from your branch into `main` on `sinhaankur/ideal-giggle`.
- **5. Wait for review and approval**. Only approved PRs are merged.

## Guidelines

- Keep changes as small and focused as possible.
- Write clear commit messages and PR descriptions.
- Do not introduce external services that break the on-device / privacy-first design.
- Follow the existing code style and structure.

## Code of Conduct

By participating in this project, you agree to be respectful and constructive. Harassment, hate, and discrimination are not tolerated.
